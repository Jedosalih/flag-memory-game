# Flag Memory Game

A fun and educational memory game that helps players learn country flags while improving memory and concentration skills.

## Features

- Multiple difficulty levels
- Interactive card-flipping animations
- Score tracking system
- Timer and move counter
- Responsive design
- Educational gameplay
- Best score saving

## How to Play

1. Start a new game.
2. Flip two cards at a time.
3. Find matching flag pairs.
4. Match all flags to complete the level.
5. Try to finish with fewer moves and in less time for a higher score.

## Difficulty Levels

### Easy
- 4 × 3 grid
- 6 flag pairs

### Medium
- 4 × 4 grid
- 8 flag pairs

### Hard
- 5 × 4 grid
- 10 flag pairs
